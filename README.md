# Contact Manager App

Android Contact Management Application built using modern Android development practices and Jetpack Compose.

## Overview

Contact Manager App is a local contact management application that allows users to create, edit, update, and delete contacts while persisting data using Room Database. The project follows MVVM architecture and uses StateFlow for reactive UI updates.

## Features

* Add Contacts
* Edit Contacts
* Delete Contacts
* View Contact List
* Local Data Persistence with Room Database
* Reactive UI Updates using StateFlow
* MVVM Architecture
* Material 3 UI with Jetpack Compose
